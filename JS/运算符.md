# 运算符

## 算术运算符

|名称|符号|名称|符号|
|:---:|:---:|:---:|:---:|
|加法|+|减法|-|
|乘法|*|除法|/|
|取模|%|增量|++变量；变量++|
|减量|--变量；变量--|负值|-值|
|转变成数字|+值|

## 字符串运算符

字符串可以通过加号（+）进行链接，如果其中一个运算数是字符串的话，另一个运算数将被转换成字符串。

##二元逻辑运算符

- 与：&&
- 或：||

## 等式运算符

- 常规的，宽松的
  - 相等：==
  - 不相等：!=
- 严格的  
  - 相等：===  
  - 不相等：!==
  
## 其它

- 前置运算符，非：!
- 排序运算符（针对字符串和数字）：>, >=, <, <=

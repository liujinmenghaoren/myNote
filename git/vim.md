# vim 编辑语法

在 git bash 中可以用 git 命令 vim 直接对文件进行编辑，下面介绍一些基础语法。

[返回目录](../README.md)

### 修改
  - :i  进入编辑状态
  - i   在光标当前位置向前插入
  - I   在本行第一个字符前插入
  - a   在光标当前位置向后插入
  - A   在本行末尾插入
  - o   向下插入一行
  - O   向上插入一行
  - :w  保存
  - :q  退出
  - :wq 保存并退出
  
 ### 删除
  - x   删除当前字符
  - dd  删除当前行
  - dw  删除当前光标下的词
  
 ### 复制粘贴
  - yy  复制当前行
  - yw  复制当前光标下的词
  - p   粘贴
  - p   粘贴在当前位置之前
